# OPUx
Tampermonkey userscript enhancer for Okoun Picture Uploader.

## Install
Use this link in Tampermonkey:

https://raw.githubusercontent.com/hanenashi/OPUx/main/opux.user.js

## Development
Edit files in `/src`, then run:
```
node build.js
```

Or push to GitHub and GitHub Actions will rebuild `opux.user.js` for you.
